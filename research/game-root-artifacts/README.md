# Game Root Research Artifacts

This folder archives scripts, extracted web chunks, reports, and scratch data that were previously placed in the game root:

`D:\steam\steamapps\common\Touhou Mystia Izakaya`

They are kept here so the plugin repository can be prepared for GitHub without mixing project research files into the original game installation.

Do not move these files back into the game folder. Runtime plugin files should be built into `bin/Release/net6.0` and copied to BepInEx by the project build target.

Files intentionally left in the game root include the game executable, Unity DLLs, `Touhou Mystia Izakaya_Data`, BepInEx runtime files, Doorstop files, save/log shortcuts, and installed plugin binaries.
